# Dados de reprovação do CEFET
## Sobre
Esse repositório tem como objetivo visualizar e gerenciar dados reais apontando problemas recorrentes e como mitiga-los. O código e o guia se encontram nesse [*notebook*](https://github.com/cassiofb-dev/bcc/blob/master/outros/reprovados/reprovados_cefet.ipynb). Todos os gráficos a seguir também foram gerados nele.
